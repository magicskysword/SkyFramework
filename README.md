# SkyFramework
Unity的简易框架
集成了xlua与FairyGUI

# 快速开始

## 1.导入Facade
将 `Assets/SkyFramework/Prefabs/SkyFrameworkFacade.prefab` 拖入场景中

## 2.编写入口脚本
继承 `ApplicationBase` 并在 `AfterInitialize()` 中编写游戏开始逻辑

# 使用的Unity插件
Addressables (版本需大于1.17.15)

# 使用的付费插件 - 库里不提供下载
Odin Inspector

# 使用的开源库
xlua

FairyGUI
